# Esquema de cores para o editor VIM - Lasmok

Coleção de esquema de cores personalizados da Lasmok para o editor Vim. Com ajustes para _flags_ BASH e C.

**Script de esquema de cores para os temas escuros:**
1. lasmok-colorido-escuro.vim
2. lasmok-neutro-escuro.vim

> Arquivo _".vimrc"_ para manter personalizações sempre que iniciar o VIM.
